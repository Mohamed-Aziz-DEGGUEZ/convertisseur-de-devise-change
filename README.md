Projet Convertisseur de Devises
1. Description du Projet
Ce projet est une application web front-end interactive, réalisée dans le cadre de l'évaluation sur les
fondamentaux du développement web.
Il s'agit d'un "Convertisseur de Devises" permettant aux utilisateurs d'obtenir les taux de change
actuels et de convertir un montant entre différentes devises, représentant un exemple de
plateforme de bureau de change.
Conformité à l'Énoncé du Projet :
- Format : Réalisé sur une seule page web.
- Technologies : HTML, CSS et JavaScript pur, sans frameworks externes.
- Objectifs : Maîtrise de la structure HTML sémantique, du design CSS responsif et de la logique
JavaScript.
2. Fonctionnalités de l'Application
Le convertisseur inclut :
- Conversion en Temps Réel via API.
- Affichage de tableaux de taux de change.
- Simulation de Spread bancaire (1%).
- Inversion rapide des devises.
- Validation des entrées et état de chargement animé.
3. Technologies et API Utilisées
- HTML5 : structure.
- CSS3 : style et responsive.
- JavaScript : logique et interactivité.
- API Frankfurter : taux en temps réel.
